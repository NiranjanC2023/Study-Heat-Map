Welcome to Focus Flow

This extension measures time on sites you label productive or distraction, optional focus sessions with Pomodoro, and shows charts — only on this device in Chrome local storage.

  • Only the active tab in your focused Chrome window counts.
  • SPA sites (YouTube, Twitter/X, etc.) update via webNavigation when the URL changes without a full page load.
  • file://, blob:, and data: URLs count as neutral time (local files aren’t matched to your host rules).
  • Internal pages (chrome://, extensions) and time when Chrome is in the background or you’re idle are not attributed to sites.
  • Use path rules in settings (e.g. youtube.com/playlist) for finer control.
  • Toolbar badge shows today’s focus % (or productive minutes) after enough classified time.
  • Quick add in the popup puts the current tab’s host on your productive or distraction list.
  • Daily & weekly goals and a session timeline live on the dashboard.
  • Focus Mode redirects distraction-list sites to a Stay Focused page; Deep Focus disables temporary overrides.
  • Tab lock pins a tab and shows a banner (best-effort — Chrome can still close tabs).
  • Pause tracking from the popup when you want a break from counting.
  • Pomodoro notifications are optional — Chrome will ask when you enable them in Settings.

Privacy summary: no accounts, no analytics server, no remote sync. Details are in PRIVACY.md in the GitHub repository.
